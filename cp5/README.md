# 🦋 The Butterflies of the Malay Peninsula
### Digital AI Companion (5th Edition, 2020)

This repository serves as the visual supplement to the definitive guide originally by **A. Steven Corbet** and **H.M. Pendlebury**, extensively revised and updated in 2020 by **George Michael van der Poorten** and **Nancy E. van der Poorten**.

This edition is a comprehensive resource covering all **1,051 species** of butterflies found in the region, featuring updated taxonomy, distribution data, and life history information.

## 📂 Repository Structure
* **`plate_XXX.jpg`**: High-resolution scans of the 132 color plates (numbered 001–132).
* **`Plate_Registry.md`**: The mapping file that allows the AI to link descriptions to these images.
